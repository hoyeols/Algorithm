## 오늘 발견한 문제 
1. 스택의 정의와 스택 top 표현하기

## 해결 방안 
1. 스택은 가장 나중에 들어온 자료를 가장 먼저처리하는 즉 후입선출 LIFO(Last In First Out)자료구조 입니다. | Operation  | 구현 | 시간복잡도 |
| -------- | -------- | -------- |
| Pop     | myList.pop()    | O(1)     |
| Push     | myList.append()     | O(1)     |
| Top     | myList[-1]     |  -    |
```
# 빈 스택(리스트) 초기화(init)
stack = []
stack

# 스택에 원소 추가(push)
stack = [1,2,3]
stack.append(4)
stack	# [1, 2, 3, 4]

# 스택에서 원소 제거(pop)
stack = [1,2,3]
top = stack.pop()
print(top)	# 3
stack		# [1, 2]

# 스택의 top 가져오기(top)
stack = [1,2,3]
top = stack[-1]
top	# 3

```
